assignment consists of 4 java files, oceanExplorer class is the main application which sets up JavaFx application and contains game UI. we created a gamemap and placed ship , two pirate ships . we handled user input using shift + up/down/left/right arrow to move the ship.the game ends when ship encounters with pirate ships, also created a reset button, allows user to restart game. OceanMap class creates oceanmap grid , it creates random map with islands, places ship. we used observer pattern to notify the pirate ships about any movement of ship. when the ship moves in desired direction and parallely pirate ship positions will be updated.notifyobservers() method is used whenever the ship's state changes, it invokes update method to each observer pirate ship and it moves accordingly.
